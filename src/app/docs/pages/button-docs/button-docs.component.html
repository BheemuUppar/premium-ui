# Overview

Buttons are the primary action control in Angular interfaces. Use them to submit forms, confirm dialogs, navigate workflows, and trigger commands. Premium UI ships pui-button as a standalone Angular component with OnPush change detection, signal-friendly inputs, and semantic tokens for light and dark themes.

Choose a variant to communicate hierarchy: primary for the main action, secondary or outline for supporting actions, and ghost for low-emphasis toolbar controls. Pair buttons with inputs and checkboxes in form layouts.
