Keyboard and screen reader support

  • Uses a native button element for keyboard activation with Enter and Space.
  • Applies visible focus styles through :focus-visible.
  • Sets aria-busy while loading and prevents duplicate activation.
  • Supports ariaLabel for icon-only controls.
  • Disabled and loading states stop events before emitting pressed.
