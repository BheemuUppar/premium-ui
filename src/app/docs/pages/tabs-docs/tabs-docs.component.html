Accessibility

  • role="tablist" on the tab list with aria-label and aria-orientation
  • role="tab" on each tab with aria-selected, aria-controls, and roving tabindex
  • role="tabpanel" on the active panel with aria-labelledby
  • Arrow keys, Home, and End move selection and focus
  • Focus ring uses shared pui-focus-ring mixin
