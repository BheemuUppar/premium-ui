Angular Command System

Premium UI ships a reusable command infrastructure — registry, search, execution, and a keyboard-first palette inspired by Spotlight, Raycast, and Linear. Register commands from any module; open the palette globally with ⌘K / Ctrl+K. By default it opens near your cursor with a macOS-style scale and fade animation. Use animation: 'smooth' for a fluid fade/slide or animation: 'minimal' for a lighter entrance.

Overview

The command system is intentionally decoupled from UI. Modules register commands through PuiCommandRegistry; the palette is one consumer. Search, grouping, recents, and execution live in services — not inside the component.

Open command palette

Registering commands

Never hardcode command arrays inside the palette component. Register from feature modules so table, admin, analytics, and docs modules can extend the system independently.

Keyboard shortcuts

Add puiCommandPaletteShortcut to your app shell. Shortcuts are configurable, SSR-safe, and cleaned up on destroy.
