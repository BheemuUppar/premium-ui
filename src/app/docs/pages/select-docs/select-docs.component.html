Accessibility

Keyboard and ARIA

Select implements the ARIA combobox pattern with listbox popup, active descendant management, and full keyboard support.

Key Action
ArrowDown / ArrowUpNavigate options
Enter / SpaceOpen panel or select active option
EscapeClose panel
TabClose panel and move focus
Home / EndJump to first or last enabled option
TypeaheadJump to matching option by label

Provide ariaLabel when the trigger has no visible label. Focus-visible styles use --pui-color-focus-ring for WCAG AA contrast in light and dark themes.

Performance

Scalable rendering

Enable virtualScroll for large datasets. The panel renders only visible rows via Angular CDK virtual scroll, keeping DOM nodes minimal even with 10k+ items.

  • Features initialize lazily — search UI renders only when searchable and open
  • OnPush change detection with signal-derived filtered options
  • Debounced async search to reduce network and render churn
  • Fixed itemHeight for predictable virtual scroll measurements
